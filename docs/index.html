csview

A fast, clean CSV viewer & editor for macOS.

Features

Everything a CSV deserves.

csview opens files instantly — even million-row ones — and lets you read, sort, search, edit, and save without ever hanging on a spinner.

⚡

Virtualized grid

Only the visible rows are rendered. Large files stream from disk with paged reads so the UI never blocks.

✎

Full editing

Click or press F2 to edit cells inline. Insert and delete rows. Cut, copy, paste a whole range from Excel or Numbers.

↶

Undo everything

Every edit, paste, insert, and delete goes on a 500-step history stack. ⌘Z and ⇧⌘Z work as you'd expect.

↕

Multi-column sort

Click a header to sort, shift-click to stack a second key. Type inference picks numeric vs. string comparison automatically.

⌕

Instant search

Live substring search across every cell with hit highlighting and row jumping. Debounced so big files stay responsive.

∑

Column statistics

Click any column header for count, unique, min, max, mean, sum, and a top-values bar chart — all computed on-demand.

◐

Six palettes

Parchment, Noir, Solarized, Ocean, Forest, Graphite — each with light and dark variants. Follows your system preference or pin it.

⊞

Multiple windows

Open one file per window. ⌘N for a blank window, ⌘O to open a new file alongside your current one.

⎇

Native menu bar

File, Edit, View, Window — real macOS menus with real keyboard accelerators. Finder double-click opens files directly.

Screenshots

Colorful, low-contrast, distraction-free.

Every palette is tuned for long reading sessions. No neon, no gradient soup, no harsh pure-white or pitch-black — just warm, editorial tones that let the data speak.

Install

Two minutes to set up.

Download the disk image and drag csview.app into Applications. Optionally symlink the binary so you can open files straight from the terminal.

1. Download & install

Grab the DMG for Apple Silicon and drop the app into ~/Applications. Because it isn't notarized, the first launch needs a right-click → Open. 

open ~/Downloads/csview_0.1.0_aarch64.dmg

2. Command line

Symlink the binary into $PATH and open any CSV with a single command. 

sudo ln -s \
  /Applications/csview.app/Contents/MacOS/csview \
  /usr/local/bin/csview

csview ~/data/employees.csv

3. File association

Right-click any .csv in Finder, choose Get Info → Open with, pick csview, then Change All… to make it the default handler.

4. Build from source

For Intel Macs or tinkering, clone the repo and run the Tauri build for your target. 

git clone https://github.com/iguana/csview
cd csview
pnpm install
pnpm tauri build
